Algorithm Theory

Understanding what each detector does under the hood.

∂x

Sobel Operator

Computes image gradients using two 3×3 kernels — one for each axis. Unlike a simple finite-difference approach, Sobel incorporates a small Gaussian smoothing step, making it inherently more noise-resistant.

Kx

-1  0 +1
-2  0 +2
-1  0 +1

Ky

-1 -2 -1
 0  0  0
+1 +2 +1
⊕

Prewitt Operator

Similar to Sobel but uses uniform weighting (no centre emphasis). Simpler to compute but slightly more noise-sensitive. Implemented manually here using cv2.filter2D with custom kernels.

Kx

-1  0 +1
-1  0 +1
-1  0 +1

Ky

-1 -1 -1
 0  0  0
+1 +1 +1
✦

Canny Algorithm

A multi-stage detector (Canny, 1986) that produces clean, thin edges. Internally: Gaussian smoothing → gradient computation → non-maximum suppression → double thresholding → hysteresis edge tracking.

  1. Gaussian smoothing
  2. Gradient magnitude & direction
  3. Non-maximum suppression
  4. Double thresholding
  5. Hysteresis tracking
